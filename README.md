# 香港陸路邊境管制站等候時間

顯示香港 8 個陸路邊境管制站的入境及出境等候時間，數據源自入境事務處開放數據平台，每 15 分鐘自動更新。

**口岸包括：**
- 香園圍口岸（Heung Yuen Wai）
- 港珠澳大橋口岸（Hong Kong-Zhuhai-Macao Bridge）
- 落馬洲口岸（Lok Ma Chau）
- 羅湖口岸（Lo Wu）
- 蓮塘口岸（Liantang）
- 文錦渡口岸（Man Kam To）
- 深圳灣口岸（Shenzhen Bay）
- 沙頭角口岸（Sha Tau Kok）

**等候級別：**
- 🟢 正常（< 15 分鐘）
- 🟡 繁忙（15-30 分鐘）
- 🔴 非常繁忙（> 30 分鐘）
- ⚪ 關閉

## 使用

直接雙擊 `index.html` 在瀏覽器開啟即可。
